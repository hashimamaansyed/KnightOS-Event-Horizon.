🛰️ KnightOS: Event Horizon
KnightOS is a high-performance desktop environment that runs entirely in your web browser. It combines a futuristic "space" aesthetic with a fast, organized interface.

🌌 The Desktop (The Singularity)
The main workspace is designed to be clean and responsive:

Selecting: Click an icon once to make it glow with a Royal Gold border.

Deselecting: Click the starry background to clear any active icons.

Window Stacking: Whichever window you click automatically jumps to the front, so your active work is never buried.

🎨 Visual Style: "Obsidian Squircles"
The icons use a specific design language called Obsidian Squircle (a smooth blend of a square and a circle):

Glass Look: Icons feature a dark, semi-transparent "frosted glass" effect.

Active Feedback: When an app is open, a gold line appears under its name.

Animations: Icons "pulse" for 2 seconds when you first launch an app.

📂 Included Applications
The Forge: A terminal for typing system commands.

The Easel: A digital painting and drawing suite.

The Abacus: A precision calculator for math.

Graphing: A tool for visualizing complex data and charts.

Horizon Browser: A built-in portal to browse the web safely.

🛠️ Controls & Shortcuts
Double-Click: Instantly opens any app.

Theme Switch: Quickly swap between Indigo (Dark) and Frosted (Light) modes.

The Horizon: The taskbar at the bottom used to switch between apps and adjust volume or brightness.

📡 Technical Foundation
Language: Built using modern web standards (JavaScript, HTML5, and CSS3).

Performance: Uses a central "engine" to handle all clicks, which keeps the system from lagging or crashing your browser.
