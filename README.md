# KnightOS-Event-Horizon.
KnightOS: Event Horizon is a futuristic web-based OS featuring a sleek glassmorphic UI. Built with a "Singularity" canvas and "Obsidian Squircle" icons, it offers multi-window multitasking, an adaptive theme engine, and a suite of apps including a terminal, paint tool, and arcade games. A premium, sci-fi portfolio and desktop experience.
